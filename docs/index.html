Focus Timer - Extensão do Chrome

Extensão inteligente para aumentar sua produtividade com timers configuráveis estilo pomodoro, notificações e interface clean.

Como usar

  1. Baixe e carregue a extensão no Chrome (modo desenvolvedor).
  2. Configure o timer facilmente através do popup.
  3. Receba notificações para ajudar no foco.
